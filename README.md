# 小紅書影片下載擴展 for Gopeed

解析並下載小紅書 (RedNote / XHS) 的**影片**與**圖集**，自動選擇最高畫質。

## 支援的 URL 格式

| 來源 | 範例 |
|------|------|
| 探索頁面 | `https://www.xiaohongshu.com/explore/{noteId}` |
| 舊版連結 | `https://www.xiaohongshu.com/discovery/item/{noteId}` |
| 短連結 | `https://xhslink.com/xxxxxx` |

## 安裝方式

1. 確認已安裝 **Gopeed 桌面版**（Web 版不支援 webview 功能）
2. 在 Gopeed 中前往 **擴展** → **安裝本地擴展**
3. 選擇此資料夾 (`gopeed-xiaohongshu/`)

## 使用步驟

1. **首次使用（必要）**：
   - 確認擴展設定中「無頭模式」為 **關閉**
   - 將任一小紅書筆記網址貼到 Gopeed 新增下載
   - 擴展會開啟瀏覽器視窗，**在視窗中登入小紅書帳號**
   - 登入後，擴展即可正常運作

2. **日常使用**：
   - 複製小紅書筆記連結（分享 → 複製連結）
   - 在 Gopeed 點「新增下載」貼上連結
   - 自動下載影片或整個圖集

3. **進階：開啟無頭模式**：
   - 登入過一次後，可在擴展設定開啟「無頭模式」
   - 之後下載時不會再彈出瀏覽器視窗

## 設定項目

| 設定 | 說明 | 預設值 |
|------|------|--------|
| 無頭模式 | 隱藏瀏覽器視窗（登入後才可用） | 關閉 |
| 偏好視訊格式 | `h264`（最相容）或 `h265`（較高畫質） | h264 |

## 注意事項

- 此擴展需要 **Gopeed 桌面版**，Web 版無法使用
- 小紅書需要登入才能完整載入影片資料
- 若下載失敗，請嘗試關閉無頭模式，確認瀏覽器視窗中已登入
- 小紅書頁面結構偶爾會更新，若持續失敗請回報

## 技術細節

擴展透過 `gopeed.runtime.webview` API 開啟內嵌瀏覽器，載入筆記頁面後從 `window.__INITIAL_STATE__` 提取媒體串流資訊，支援 H.264 / H.265 格式，並自動選取最高解析度串流。
